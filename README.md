# Python おみくじ

初心者向けの小さな Python プログラムです。

ターミナルで実行すると、「大吉」「中吉」「小吉」「末吉」のどれかをランダムに表示します。結果に合わせた短いメッセージも出ます。

## ファイル構成

- `omikuji.py`: おみくじプログラム本体
- `docs/`: ブラウザで遊べるWeb UI版
- `README.md`: 使い方の説明

## Python版の実行方法

このフォルダで、次のコマンドを実行します。

```powershell
python .\omikuji.py
```

もし `python` コマンドが使えない場合は、Pythonをインストールするか、環境に合わせたPython実行コマンドを使ってください。

## Web UI版で遊ぶ

GitHub Pagesで公開しています。

[おみくじポケットで遊ぶ](https://oliverblackwood71.github.io/omikuji-python-example/)

ローカルで開く場合は、次のファイルをブラウザで開きます。

```text
docs/index.html
```

Web UI版は、Python版と同じ「ランダムに結果を選ぶ」という考え方を、HTML/CSS/JavaScriptでブラウザ向けに作り直したものです。

- HTML: 画面の部品
- CSS: 見た目とゲーム風の雰囲気
- JavaScript: ボタンを押したときの動き
- Web Audio API: BGMと効果音をコードで鳴らす仕組み
- localStorage: 引いた回数、今日のベスト、最近の結果をブラウザに保存する仕組み
- Three.js: 背景で回る3Dおみくじ箱を描くJavaScriptライブラリ

ボタンを押すと、おみくじ箱が揺れて、札が飛び出し、結果カードが光るようにしています。画像素材は使わず、HTML/CSS/JavaScriptだけで作ったアニメーションです。

背景の3Dおみくじ箱はThree.jsで描いています。これは新しいプログラミング言語ではなく、JavaScriptから3Dの形、光、カメラ、回転を扱いやすくするライブラリです。

音楽は既存曲を使わず、JavaScriptで生成した短いオリジナルのレトロ風BGMです。ブラウザの自動再生制限があるため、音を鳴らすには画面右上の `BGM OFF` ボタンを押してください。

記録はブラウザ内に保存されます。ページを閉じても続きから遊べますが、別のブラウザや別の端末には引き継がれません。最初からやり直したいときは、画面下の「記録をリセット」を押してください。

```text
結果: 大吉

もう一度引きますか？ (y/n): n

また遊んでください。今日も良い一日を。
```

## 何をしているコードか

`random.choice()` を使って、複数のおみくじ結果の中から1つをランダムに選んでいます。

```python
return random.choice(fortunes)
```

`fortunes` は、結果とメッセージのセットを入れたリストです。

```python
("大吉", "今日は新しいことを始めるのにぴったりです。小さく一歩進めてみましょう。")
```

このようなセットを増やすと、おみくじの種類を増やせます。
